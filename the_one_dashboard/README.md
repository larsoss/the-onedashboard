# The-One Dashboard

A HomeKit-style Home Assistant dashboard. Glassmorphism tiles, real-time updates, per-user settings — no configuration needed.

![Home view](https://raw.githubusercontent.com/larsoss/the-onedashboard/main/docs/screenshots/home.jpg)

---

## Installation

1. Go to **Settings → Add-ons → Add-on Store**
2. Click **⋮ → Repositories** → add `https://github.com/larsoss/the-onedashboard`
3. Find **The-One Dashboard → Install → Start**
4. The dashboard appears in your HA sidebar automatically

---

## Features

- **Tiles** — lights, switches, thermostats, locks, covers, sensors, scenes, automations, scripts, weather, media players, cameras, calendars, person cards
- **Edit mode** — drag to reorder, resize tiles (1×1 up to 4×2), pick icons, favorite or hide entities
- **Per-user settings** — each HA user gets their own layout, favorites, and theme, synced across devices
- **Color moods** — iOS Dark, Cyber Glass, Ocean, Midnight and more; fine-tune accent color, opacity, and neon glow
- **Sidebar** — clock, weather, active device counts, HA notifications
- **Room backgrounds** — set a custom photo per area card
- **Kiosk-friendly** — `?view=woonkamer` deep-links to an area; `?menu=false` hides the tab bar
